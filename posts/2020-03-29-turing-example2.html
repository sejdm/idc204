Turing machine example: Power of 2

29 March, 2020

In this example, we design a Turing machine to check whether the number of 0s in a string of 0s is a power of 2.

\(\mathbf{q_0}\): If you read a 0, then proceed to \(q_1\) otherwise reject.

_ 0 X
\(q_0\) (Reject, _, L) (\(q_1\), _, R) (Reject, _, L)

\(\mathbf{q_1}\): To halve the number of 0s left, keep sweeping to the right until you hit a 0 or blank (i.e.Â move right but remain in the same state on seeing an X). If you hit a 0, shift to the state \(q_2\) which is meant to denote that even number of 0s were read.

_ 0 X
\(q_1\) (Accept, _, R) (\(q_2\), X, R) (\(q_1\), X, R)

\(\mathbf{q_2}\):

_ 0 X
\(q_2\) (\(q_4\), _, L) (\(q_3\), 0, R) (\(q_2\), X, R)

\(\mathbf{q_3}\):

_ 0 X
\(q_3\) (Reject, _, R) (\(q_2\), X, R) (\(q_3\), X, R)

\(\mathbf{q_4}\):

_ 0 X
\(q_4\) (\(q_1\), _, R) (\(q_4\), 0, L) (\(q_4\), X, L)

Here is the final transition table:

_ 0 X
\(q_0\) (Reject, _, L) (\(q_1\), _, R) (Reject, _, L)
\(q_1\) (Accept, _, R) (\(q_2\), X, R) (\(q_1\), X, R)
\(q_2\) (\(q_4\), _, L) (\(q_3\), 0, R) (\(q_2\), X, R)
\(q_3\) (Reject, _, R) (\(q_2\), X, R) (\(q_3\), X, R)
\(q_4\) (\(q_1\), _, R) (\(q_4\), 0, L) (\(q_4\), X, L)
